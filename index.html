    Property searcher utilizing Breadth First Search Algorithm

    This small project is built to imitate BFS search algorithm. Data is stored in graph, with BFS logic and interactive UI combined this wonderful application. Using Python, TKinter, and BFS Algorithm, user can input location, price, property type, to search for their desired property.
